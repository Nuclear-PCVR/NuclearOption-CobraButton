# NuclearOption-CobraButton
A mod for Nuclear Option that adds a Cobra / AOA limiter button that disables your plane's G-Limiter, allowing extreme AOA maneuvers to be performed at high-speeds.
Mod is not to be blamed for any missing wings.
Works best when flying an Ifrit, but also works (with risk of wings flying off) in most other planes too full list of planes tested below.
CI-22 Cricket: Does nothing, as the cricket can't realistically hit its G-limit anyway.
T/A-30 Compass, EW-25 Medusa, SFB-81 Darkreach, Alkyon AB-4: Works, but frequently causes wing failure at higher speeds.
A-19 Brawler: Works, but doesn't significantly increase maneuverability.
FS-12 Revoker & FS-20 Vortex: Works, but prone to losing control.
KR-67 Ifrit: Works flawlessly; grants an extreme maneuverability bonus when anywhere below mach-1, above mach-1 pilot passes out near-instantly and wings are semi-prone to failure.
Any Helicopter: Does nothing, as helicopters don't really need to worry about the G-limit.

Disclaimer: The code responsible for allowing in-game control mapping was pulled straight from the BOTE mod (https://github.com/MinecrackTyler/BoscaliOceanTrainingExercise), all credits to its creator for that section of the code.
