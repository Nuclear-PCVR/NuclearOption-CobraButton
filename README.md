# Nuclear Option - Cobra Button Mod
A mod for Nuclear Option that adds a Cobra / AOA limiter button that disables your plane's G-Limiter, allowing extreme AOA maneuvers to be performed at high-speeds.<br/>
Mod is not to be blamed for any missing wings.<br/>
Works best when flying an Ifrit, but also works (with risk of wings flying off) in most other planes too full list of planes tested below.<br/>
CI-22 Cricket: Does nothing, as the cricket can't realistically hit its G-limit anyway.<br/>
T/A-30 Compass, EW-25 Medusa, SFB-81 Darkreach, Alkyon AB-4: Works, but frequently causes wing failure at higher speeds.<br/>
A-19 Brawler: Works, but doesn't significantly increase maneuverability.<br/>
FS-12 Revoker & FS-20 Vortex: Works, but prone to losing control.<br/>
KR-67 Ifrit: Works flawlessly; grants an extreme maneuverability bonus when anywhere below mach-1, above mach-1 pilot passes out near-instantly and wings are somewhat prone to failure.<br/>
Any Helicopter: Does nothing, as helicopters don't really need to worry about the G-limit.

### How to use mod
Step 1: Bind the "Cobra" button (inside the normal control binds menu) to your button of choice<br/>
Step 2: Get in a plane, get up to speed, hold down the button you set for about a second, and pull up, if everything worked correctly, you should be able to pull up significantly more than normal.

### How exactly the mod works
Normally, every plane in the game has a set G-Limiter (for example, the Ifrit's is 9G), regardless of how hard you try to pull up, the plane will actively reduce your control to prevent you from going over the limit.<br/>
However, when you press this mod's button, this limit is temporarily massively increased (limit, as far as I could see, cannot be fully removed), allowing you to pull 20G+ maneuvers.
